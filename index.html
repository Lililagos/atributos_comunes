Aributo translate

El atributo lang indica el idioma del contenido de la etiqueta. El valor de dicho atributo tendrá que ser el código ISO 639-1 del idioma al que hace referencia. El atributo translate tambié se usa para indicar si el contenido de un texto es o no traducible: CSS, HTML, ID, CLASS entre otras.

Así, indicamos al navegador o a herramientas como Google Translator, que esa frase específica no se debe traducir porque queremos que aparezca siempre. Si no se indica el atributo translate en una etiqueta, es como si tuviera el valor yes por defecto.
